# Dannyflix — Welcome Page

Onboarding site for Dannyflix (Plex) users. Covers getting started, requesting content, tips, and other available services.

Built with Vue 3 + Vite. Deployed to GitHub Pages via GitHub Actions.

## Development

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` deploy automatically via GitHub Actions. To deploy manually:

```bash
npm run deploy
```
